    Dictionary

    Creating a dictionary

    {% highlight py %} emp1 = {'name': 'Bob', 'job': 'waiter', 'salary': 3} {% endhighlight %}
    {% highlight py %} emp1 = dict(name='Bob', job='waiter',salary=3) {% endhighlight %}
    {% highlight py %} emp1 = {} emp1['name'] ='Bob' emp1['job'] = 'waiter' emp1['salary'] = 3 {% endhighlight %}

    Nesting in a dictionary

    {% highlight py %} emp1 = {'name': {'first': 'Bob', 'last': 'Smith'}, 'jobs':['waiter', 'host'], 'salary': 3 {% endhighlight %}
