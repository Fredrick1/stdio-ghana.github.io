File object

open function creates a Pythong file object which links to a file on your machine

Once open, strings of data can transferred to and from the file

To open a file:

{% highlight py %} afile = open(filename, mode) afile.method {% endhighlight %}

The mode can be 'r' for read, 'w' for write, or 'a' to append

Reading csv files

{% highlight py %} import csv import sys f = open('file path', 'r') #input the file path for the PracticeFile.csv try: reader = csv.reader(f) for row in reader: print row finally: f.close() {% endhighlight %}

Writing to csv files

{% highlight py %} import csv import sys f = open('file path', 'w') #input the file path for the PracticeFile.csv try: writer = csv.writer(f) writer.writerow( ('Title 1', 'Title 2', 'Title 3') ) for i in range(10): writer.writerow( (i+1, chr(ord('a') + i), '08/%02d/07' % (i+1)) ) finally: f.close() print open('file path', 'r').read() #input the file path for the PracticeFile.csv {% endhighlight %}

Reading csv as dictionary

{% highlight py%} import csv import sys f = open('file path', 'r') try: reader = csv.DictReader(f) for row in reader: print row finally: f.close() {% endhighlight %}

Writing dictionary to csv

{% highlight py %} import csv import sys f = open('file path', 'w') try: fieldnames = ('Title 1', 'Title 2', 'Title 3') writer = csv.DictWriter(f, fieldnames=fieldnames) headers = dict( (n,n) for n in fieldnames ) writer.writerow(headers) for i in range(10): writer.writerow({ 'Title 1':i+1, 'Title 2':chr(ord('a') + i), 'Title 3':'08/%02d/07' % (i+1), }) finally: f.close() print open('file path', 'r').read() {% endhighlight %}
